ANGULAR-AJAX-EXAMPLE
########################################

Приклад get запит на сервер і відповідь у вигляді json який ловить angular і відображається на сторінці за допомогою angular

Installation
============
Встановити node js, після чого в командій стрічці (cmd) ввести наступні команди. 
* npm install
* node server.js
